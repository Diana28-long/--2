# Python Input Validation Project

## Опис проєкту
Це простий консольний застосунок на Python, який дозволяє користувачу вводити число та перевіряє коректність введення.

Програма обробляє помилки та не дозволяє ввести некоректні дані.

---

## Функціонал
- ✔ введення числа
- ✔ обробка помилки введення
- ✔ перевірка коректності даних
- ✔ автоматичне тестування (assert, unittest, pytest)
